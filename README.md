# KYH:s IT-grundkurs

Detta är en interaktiv webbaserad kurs för nybörjare som förklarar grundläggande IT-koncept.

## Innehåll

- `index.html` – kursens huvudsida med landningssida och moduler
- `style.css` – styling för färger, layout och responsiv design
- `script.js` – kursens interaktivitet, modulnavigering, quiz och progresshantering

## Funktioner

- Landningssida med kursöversikt och startknapp
- Åtta moduler med teori, exempel och quiz
- Progressbar som sparar slutförda moduler i `localStorage`
- Enkla knappar, hover-effekter och responsiv layout

## Så använder du projektet

1. Öppna `index.html` i din webbläsare.
2. Navigera mellan modulerna i vänstermenyn.
3. Klicka på knapparna för att gå vidare och svara på quiz-frågorna.
